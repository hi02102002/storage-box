# storage-box
A simple storage to use save file on cloud.
# Features
- Login 
- Create folder / Upload file in to current folder
- Nest folder
- Move folder / file to trash
- Remove sigle file / folder or remove all in trash
# Technologies
- React 
- Vite
- Firebase
- Tailwindcss
